Projects

Project AT (Application Tracker)

Technologies - Node.js, React.js, AWS RDS Cloud Database (MySQL), HTML, CSS, JavaScript, Bootstrap

Project AT was our Computer Science Capstone project. The idea behind the application was to design an all-in-one application to track jobs during the career searching process. We also included an interview prep page so the job applicant can save notes and interview information about a job they are looking to land. While we didn't complete everything we wanted to, we were able to present our product in front of a small audience at one of our university's lecture halls.

BattleShip Game

Technologies - HTML, CSS, JavaScript, Bootstrap

One of our projects in our Web Application Development class was to make a functioning battleship game. The game board is set up with a random placement of boats. The player starts the game by specifing the max amount of misses before they lose the game. The player can click the board and try to sink all ships before running out of misses. The application also keeps track of the gameplay statics (such as shot count, hit count, etc.).

ALDI PTO Project

Technologies - Node.js, Pug (Jade), phpMyAdmin Database (MariaDB), HTML, CSS, JavaScript, Bootstrap

The PTO Tracker was done in partnership with ALDI USA. The goal for this project was to design a web application that would track vacation, sick, and personal days that associates, managers, and directors can take. Our application also would allow associates to make PTO requests and have their manager approve/reject them within the application as well. We were able to work with members of ALDI USA as our clients. In the end, we were able to present to them a functioning tracker with request and approve/reject capabilities. While there is still room for improvement, we were happy with what we were able to accomplish in the short time-frame.
